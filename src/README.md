### How to run a C program


#### ```#Linux```


[1]. Install ```GCC```.

    sudo apt install gcc
    
   >To check the installation 
   ```gcc -v```
   
[2]. Clone the repo and navigate to the desired folder.

[3]. Open terminal and type.

    gcc <filename.C>

> For example -: ```gcc main.c```
